This Website

As my ability grows, so does my portfolio website. One could say we evolve together. I am constantly adding new elements, redesigning its components, trying new color schemes, tinkering with animations, etc. From its first iteration to what you are seeing now, it truly represents my journey as a software developer.

This website features a number of fun features. I utilized JavaScript to build slide-in animations that activate when the user scrolls to specific locations on a page. Combining CSS effects with shades of blue I created a ‘neon glow’ effect. I also create my own icons, such as my logo/home-button, using Adobe Illustrator.
