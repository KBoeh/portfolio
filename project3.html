Museum Experience App

Working on this was an amazing experience. I worked with people all over the world, from Korea to Nigeria. I worked on 2 remote teams of software designers and developers. We were led by a project manager and supported by technical managers.

We collaborated through Asana and utilized Git and GitHub to submit branch requests to the project's main repository. I styled responsive pages for mobile to desktop and learned to implement React Hooks to fetch API data.

Tech Stack: React.js, React Hooks, Node.js

Status: This project was a bootcamp educational experience and development was halted at the end of the camp's second phase.
