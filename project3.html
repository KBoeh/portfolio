E-Commerce Website

A UI/UX designer and I volunteered to design and build the More Knickknacks e-commerce website for a business owner who sells vintage items. Utilizing input from the owner, we developed an interface and theme that delivers a pleasant user experience. The site sports an Americana vintage color scheme and a custom logo design that is based on the origin story of More Knickknacks.

I built the site to be fully responsive so it delivers an enjoyable user experience on mobile to desktop screen sizes. I included fun animations such as a CSS only mobile hamburger button that transitions into an ‘X’ on-click and on-hover overlays that reveal buttons and text. Future functionality will include a customer login so purchases can be tracked, favorite lists can be saved, etc. Also, no e-commerce site is complete without a working shopping cart with the ability to make purchases.

Status: currently under development.
