Designer Portfolio

I built this portfolio for a UI/UX designer. I created this website according to the Figma style guides and prototypes provided by the designer. I used a mobile first approach and built responsive pages that look good on all screen sizes.

One feature I really enjoyed building was the mobile navigation button. When the "hamburger" button is pressed it transforms into an X and transitions back when pressed again. Also, when the navigation menu slides into view, a sleek tinted overlay covers the content under the menu. Creating a smooth and visually pleasing user experiences is always a fun challenge.

Status: currently under development.
